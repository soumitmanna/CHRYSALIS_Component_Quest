# User Manual

## Starting the Application

1. Open terminal
2. Activate virtual environment

venv\Scripts\activate

3. Start Flask

python app.py

4. Open browser

http://localhost:5000

## Uploading Components

1. Click Upload Component Image
2. Select component image
3. Click Detect Component

## Dashboard

The dashboard displays:

* Farm Brain Status
* Water Guardian Status
* Completion Percentage

## Completion

When both components are detected:

* Farm Brain Unlocked
* Water Guardian Unlocked
* Smart Farm Completed
